FataleFlix

This is a simple API built as a resource for film buffs looking for their next unsettling & woman-centered screening.

Version Date Changes version 1 1/1/2020 initial deployment

Allows new user to update their profile. /users/[username] PUT A JSON object that holds data about the current user to be update with new information.

{
"Username": "newusername",
"Password": "newpassword",
"Email": "newemail@gmail.com",
"Birthday": "1999-01-21",
}

 A message indicating whether or not the user's profile has been successfully changed.
Allows user to add a movie to a queue. /users/[username]/queue/[filmID] POST A JSON object holding data about the film to be added to the user's queue. A message indicating whether the film was added to the user's queue.
Allows user to remove a movie from their queue. /users/[username]/queue/[filmID] DELETE None A text message indicating whether the film was successfully removed from the user's queue.
Allow existing user to deregister. /users/[username] DELETE None A message indicated whether the user's account was successfully removed from the site.
